Waterbot

NodeJS-based modular IRC bot.

Download as .zip Download as .tar.gz View on GitHub

What does it do?

Stuff like this:

<JoeUser> so I was testing issue #86 the other day, and I can't reproduce it
<waterbot> Issue 86: Crash on trying to open binary files. https://github.com/some/repo/issues/86
<BobNerd> yeah, I couldn't either -- he's using an old version, I think #51 fixed that
<waterbot> Pull 51: Handle multiple encoding schemes. https://github.com/some/repo/pull/51

Besides supporting the GitHub issue system, there's modules for any Trac site or Google Code. The bot is hosted on Heroku and operates on Freenode and Mozilla networks.
